# Learn-ECMAScript6

 ES6的新特性如下

- block scoping － 使用 let 、const 作用域，块辖域（第一部分ch01）
- classes - 各种 ‘类’，再也无需用 CoffeeScript 的语法糖写类了（第二部分ch02）
- collections － 集合、映射、弱集合、弱映射（第三部分ch03）
- generators - 未来的.js 代码中将有无数生成器，不学一点就看不懂 JS 代码了（第四部分ch04）
- promises － 用标准化了的方法进行延迟和异步计算（第五部分ch05）
- symbols － 唯一的、不可修改的数据（第六部分ch06）
- arrow functions － 箭向函数（第七部分ch07）
- template strings － 模板字串（第八部分ch08）
